Strong Password API

Available only on the AWS Marketplace

This service provides a serverless API for generating and validating passwords. Use strongpassword.io to create passwords and validate that existing passwords meet complexity requirements.

The API supports a number of parameters that specify how the password should be created (e.g. length, whether it should contains letters, numbers and symbols, and so on). The validator service checks whether the supplied password is strong enough using default or user-supplied policies.
